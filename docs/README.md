# Landing Page Interactiva — Programación Orientada a Objetos (POO) en Java

Proyecto académico desarrollado como material interactivo de apoyo para el aprendizaje de Programación Orientada a Objetos (POO) en Java.

La aplicación consiste en una landing page educativa que presenta de forma organizada, visual e interactiva los conceptos fundamentales de la Programación Orientada a Objetos, incluyendo teoría, ejemplos prácticos y ejercicios.


## Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript (main)
- Tailwind CSS
- Lucide Icons

## Arquitectura del Proyecto

El contenido se genera dinámicamente mediante JavaScript utilizando un arreglo de objetos (`topics`) que almacena:

- Información conceptual
- Ejemplos de código
- Ejercicios
- Navegación interna

Cada tema es renderizado dinámicamente en el DOM para facilitar mantenimiento y escalabilidad.


## Metodología de Desarrollo

El proyecto fue desarrollado siguiendo principios de:
- Organización modular
- Separación de responsabilidades
- Reutilización de componentes
- Aprendizaje progresivo de POO
